A Bloomberg Terminal-inspired personal website built with Next.js, TypeScript, and Tailwind CSS. Features a terminal boot sequence, real-time clock, market ticker, and a full markdown-based content system.

## Tech Stack

- Next.js 14+ with App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Markdown with gray-matter + unified/remark/rehype

## Features

- Terminal boot animation on first visit
- Bloomberg-style dashboard with live widgets
- Full content management via markdown files
- BD Sterling Capital Management branding section
- Sterling Empire entity showcase
- Responsive design with terminal aesthetic
